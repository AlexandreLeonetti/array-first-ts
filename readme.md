# Get first n elements using TypeScript 

> Get first n elements using TypeScript. 

## Install

```sh
npm install @leonetti/array-first-ts
```

## Usage

```js
import {arrayFirst} from '@leonetti/array-first-ts';

arrayFirst(['a', 'b', 'c', 'd', 'e', 'f']);
//=> 'a'

arrayFirst(['a', 'b', 'c', 'd', 'e', 'f'], 4);
//=> ['a', 'b', 'c','d']

```
